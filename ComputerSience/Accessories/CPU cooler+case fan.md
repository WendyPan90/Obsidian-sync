# 關注參數

#### 風冷 :

![[Pasted image 20230728141315.png]]

1. 熱管數量
2. 扣具兼容性
3. 散熱氣尺寸
4. 風扇參數

#### 水冷 :

![[Pasted image 20230728141428.png]]

1. 冷排尺寸
2. 扣具兼容性
3. 冷頭 (水泵) 尺寸
4. 風扇參數
5. 保修時間 & 保修政策

>==**要注意的是**== : 越高端的主機板越容易和散熱器衝突，因馬甲(包裝)尺寸較大
