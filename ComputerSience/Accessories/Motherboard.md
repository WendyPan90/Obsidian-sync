# 常見主機板板型

1. **ATX (大)**
	* 擴展性強
	* 體積大
	* 價格相對較高
2. **mATX (中)**
	* 性價比高
	* 體積適中
	* 擴展性好
3. **ITX (小)**
	* 體積最小
	* 價格相對較高

(有特大但一般不會用到)

# 關注參數

![[Pasted image 20230728121037.png]]

1. 版型
2. CPU接口(針腳)
3. 芯片組
4. IO接口
5. 內存&硬體插槽 (數量)
6. 供電接口&設計
7. PCI-E接口
8. 散熱
9. Wi-fi (是否有wifi無線網卡)
>==要注意的是== : CPU和主機板的相容性
>>intel配intel / AMD配AMD
>>Intel常見CPU接口 : LGA 1700、LGA 1851(之後可能會升級)
>>AMD常見CPU接口 : AM4、AM5(之後可能會升級)
